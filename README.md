# 🔮 iChing · 周易占卜 & 答案选择器

一款轻量、纯净、完全离线的跨平台工具应用，提供《周易》六爻占卜模拟和自定义答案随机抽取功能。  
**本应用仅供娱乐与传统文化学习，请理性看待结果，相信科学。**

---

## ✨ 主要功能

### 1. 周易占卜模拟
- 模拟传统三枚硬币起卦（花=3，字=2），随机生成六爻
- 自动计算本卦、变卦，并标注变爻位置（老阴变阳、老阳变阴）
- 显示六爻分数（6/7/8/9）及爻线（阳爻 `===`，阴爻 `= =`）
- 一键复制标准提示词，可粘贴至 AI 对话工具获取深度解读
- 支持 **摇一摇** 快速占卜（可开关）
- 震动反馈（按钮点击、摇一摇触发）

### 2. 答案选择器
- 自定义 2/3/4 个选项，输入任意文本
- 老虎机滚动动画，随机抽取一个选项
- 震动反馈（仅 App 端）
- 一键复制抽取结果

### 3. 隐私与安全
- 完全离线运行，不联网
- 无需任何系统权限（摄像头、存储、位置等）
- 不收集任何个人信息，不集成任何第三方 SDK
- 提供完整的隐私政策说明

## 📦 安装与运行

### 环境要求
- HBuilderX 3.8+（推荐）
- Node.js 16+（可选，用于 CLI 构建）

### 步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/your-repo/iching.git
   ```

2. **用 HBuilderX 打开项目**
   - 文件 → 导入 → 从本地目录导入 → 选择项目根目录

3. **运行**
   - 运行 → 运行到浏览器 / 运行到手机 App 基座 / 运行到微信小程序

4. **发行（打包）**
   - 发行 → 原生 App-云打包 → 填写证书（Android / iOS）
   - 发行 → 小程序-微信 → 上传至微信公众平台
